Analyse an Article About the Project

What the App Does

🔍

Fetch & Score Comments

Paste any Wikipedia URL and the app fetches talk page comments via the MediaWiki API, then scores each one for toxicity using your chosen ML model.

🤖

Multiple ML Models

Compare predictions from Lasso Logistic Regression, SVM, Gradient Boosting, and more — all trained on the Jigsaw toxicity dataset.

⚠️

Configurable Thresholds

Set an auto-ban threshold and a manual-review threshold. Comments above the first are flagged automatically; those in between are queued for human review.

📊

Exploratory Analysis

Dive into the dataset statistics, word distributions, toxicity trends over time, and model performance metrics on the EDA and Dashboard pages.

🔀

Sort Demo

A subtle nod to our team name. Watch bogosort and merge sort race each other in an animated visualisation, illustrating why algorithm choice matters.

📈

Model Evaluation

Inspect per-model metrics (accuracy, precision, recall, F1, AUC) and confusion matrices on the Nerdy Dashboard to understand model trade-offs.

How It Works

  1. 1
    Enter a Wikipedia URL

    The app fetches talk-page comments for that article from the MediaWiki API.

  2. 2
    Choose a Model & Thresholds

    Select which trained classifier to use and tune the auto-ban / review score cutoffs.

  3. 3
    Analyse Comments

    Each comment is vectorised and scored. Results appear instantly in an interactive grid.

  4. 4
    Review & Act

    Drill into individual comments to see token-level toxicity highlights and take moderation actions.
